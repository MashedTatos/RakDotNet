# RakDotNet

RakNet for the .NET platform.

## Features

- Full-featured* server implementation
- Asynchronous
- Extensible
- [TCP/UDP Protocol](https://bitbucket.org/lcdr/raknet_shim_dll/wiki/Home) support
- 100% C#

*\* ORIGINAL RakNet protocol is NOT implemented*
